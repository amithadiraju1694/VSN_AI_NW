About this App.

Thank you for interest in using our app. This app is used to predict next word , based on some part of sentence, from holy text Vishnu Sahasranamam. Input to this app must be in english script ( alphabets ) but with sanskrit meaning. This app uses NLP and deep learning to predict the next word in real-time.

Examples of Using this App

  • 1. Input a single sentence

    īśānaḥ prāṇadaḥ prāṇo

  • 2. Input more than one sentence

    īśānaḥ prāṇadaḥ prāṇo | bhū-garbho mādhavo
